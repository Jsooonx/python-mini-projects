# Python Mini Projects

A collection of small Python projects created for learning, experimentation, and fun.

This repository contains various **mini programs and visualizations** built with Python.
Each project focuses on practicing different programming concepts such as:

* algorithms
* simulations
* simple games
* data visualization
* problem solving

The goal of this repository is to **learn by building small projects consistently**.

---

## Projects

Some examples of projects in this repository include:

* 🐍 Snake Game (Pygame)
* 📊 Sorting Algorithm Visualizer
* 🔢 Small simulations and algorithm experiments
* 🎨 Visualization experiments with Python libraries

Each project is stored in its **own folder or script**.

---

## Requirements

Most projects use standard Python libraries.
Some projects may require additional packages such as:

```bash
pip install pygame matplotlib numpy
```

Check the individual project files if extra dependencies are needed.

## How to Run

Clone the repository:

```bash
git clone https://github.com/yourusername/python-mini-projects.git
```

Navigate to a project and run it with Python:

```bash
python project_name.py
```
